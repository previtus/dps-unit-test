The generated target signatures in this folder are to be used for detection of point source events in EMIT data, however not for quantification - please refer to the publication for more details. Shortly, we have generated these targets using MODTRAN by simulating spectral radiance with and without the gas. However, these enhancements relate to emission rates via an unknown windspeed-dependent scale factor. We note that the generated matched filter product might need to be scaled (as is done before using them in the machine learning models in pipeline/B_ml_segmentation/run_models_other_gases.py - and also when scoring by using different threshold for selection of background pixels in pipeline/C_plume_scoring/scoring_gas_utils.py)
